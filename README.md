# Weather-Diary

**Автор:** [Артём Шигапов]

## Описание
Приложение для ведения личного дневника погоды. Позволяет фиксировать дату, температуру, описание и наличие осадков. Данные сохраняются локально в формате JSON.

## Функционал
- Валидация даты и числового значения температуры.
- Фильтрация записей по минимальной температуре.
- Автоматическое сохранение и загрузка данных.

## Инструкция по запуску
1. Установите Python 3.8 или выше.
2. Скачайте файл `weather_diary.py`.
3. Запустите приложение командой:
   ```bash
   python weather_diary.py
   ```

## Примеры тестов
1. **Тест валидации**: Попробуйте ввести в поле температуры текст вместо числа — программа выдаст предупреждение.
2. **Тест фильтра**: Добавьте записи с температурами +5, +15, +20. Введите в фильтр "10" — записи ниже +10°C исчезнут из списка.
